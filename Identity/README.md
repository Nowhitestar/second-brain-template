# Identity

This directory contains your personal identity profile — who you are, what you're doing, and how you prefer to work.

## Members

| File | Purpose |
|------|---------|
| TELOS.md | Mission, goals, and values |
| CONTEXT.md | Current state and active focus |
| PROFILE.md | Contextual preferences and communication style |

## Rules

- All changes to files in this directory require **user approval** (Type B change)
- New preferences go through the candidate pool first (`System/working-memory/candidates.md`)
- Only promote to PROFILE.md after user confirmation
