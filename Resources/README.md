# Resources

Reference materials — articles, reports, research, notes. Raw materials that get woven into Wiki/.

Each file should have frontmatter:

```yaml
---
title: Article Title
source: URL or origin
date: YYYY-MM-DD
tags: [tag1, tag2]
processed: false        # true after woven into Wiki
wiki_ref: [[entity]]    # link to Wiki page after processing
---
```
